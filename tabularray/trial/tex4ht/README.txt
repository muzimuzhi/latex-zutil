
How to improve tabularray support for TeX4ht

1. Download tex4ht-4ht.tex and tex4ht-html4.tex

2. Copy tabularray code in these files into tabularray-4ht.tex

3. Modify tabularray-4ht.tex trying to improve it

4. Run "texlua tex-to-4ht.lua" to generate tabularray.4ht

5. Compile test documents with the new tabularray.4ht

6. Submit the code in tabularray-4ht.tex to TeX4ht maintainers
